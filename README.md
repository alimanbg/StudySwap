# StudySwap – Textbook & Study Resource Marketplace

A peer-to-peer platform for students to buy and sell used textbooks and study materials.

## Live URL
[https://your-site.com](https://your-site.com)

## Setup Instructions
1. Clone this repository.
2. Import `database/schema.sql` into your MySQL database.
3. Copy `includes/config-sample.php` to `includes/config.php` and update database credentials.
4. Place the project in your web server's document root (e.g., `htdocs`).
5. Access via `http://localhost/study-swap`.
